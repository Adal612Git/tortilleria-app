> Genial que ya quedó. Te dejo el checklist diario y atajos útiles.

  Inicio diario (sin tocar BD)

  - Abrir el proyecto: cd ~/Documentos/tortilleria-app
  - Levantar PHP: php artisan serve --port=8001
  - Abrir en navegador: http://127.0.0.1:8001/login
  - Si vas a editar front, en otra terminal: npm run dev (si no, puedes
  omitirlo)

  Cuando la BD ya existe (lo normal)

  - No hay que migrar ni seedear cada día. La BD SQLite persiste en database/
  database.sqlite.

  Solo si la BD no existe o la reiniciaste

  - Crear archivo: touch database/database.sqlite
  - Migrar y sembrar: php artisan migrate --seed

  Extras útiles (opcional)

  - Procesar jobs (si los usas): php artisan queue:work (en otra terminal)
  - Si Vite dice “Port in use”, usa el que muestre (5173, 5174, etc.)
  - Si PHP 8001 está ocupado, usa otro: php artisan serve --port=8002
  - Rellenar catálogos (empleados/tiendas/proveedores): php artisan db:seed --class=Database\Seeders\EmployeesStoresSuppliersSeeder

  Solución rápida de problemas

  - Cambios de rutas/config no se reflejan: php artisan route:clear && php
  artisan config:clear && php artisan view:clear
  - Bloqueo por intentos de login (throttle): php artisan cache:clear
  - Usuarios de prueba únicamente: php artisan db:seed --class=Database\
  \Seeders\\TestUsersSeeder

  ¿Quieres que agregue un script bin/start-dev.sh que haga chequeos (BD, cachés)
  y arranque todo con un solo comando?
